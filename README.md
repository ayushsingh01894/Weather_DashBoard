# 🌦️ Weather Dashboard

A modern responsive Weather Dashboard built using HTML, CSS and JavaScript with OpenWeather API.

🔗 Live Demo:  
https://ayushsingh01894.github.io/Weather_DashBoard/

---

## ✨ Features

- 📍 Auto Location Detect
- 🌙 Dark / Light Mode Toggle
- 🌡️ Celsius / Fahrenheit Toggle
- ⏱️ Hourly Forecast
- 📅 5 Day Forecast
- 📊 24 Hour Temperature Chart
- 🌅 Sunrise & Sunset Time
- 🎨 Weather Based Background
- 📱 Fully Responsive Design
- ⚡ Loading Animation

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Glass UI)
- JavaScript (Vanilla)
- Chart.js
- OpenWeather API
